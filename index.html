Implementation Guide

01 Environment Setup

Sebelum memulai, pastikan Node.js (LTS Version) sudah terinstal di OS Anda. Framework ini menggunakan engine Puppeteer dengan lapisan Stealth untuk menghindari deteksi sidik jari digital.

Step 1: Inisialisasi & Install Dependencies

npm init -y && npm install puppeteer puppeteer-extra puppeteer-extra-plugin-stealth
COPY

02 Feature Functionality

📍 Selector Strategy

  • • CSS/XPATH: Gunakan untuk elemen statis. Ambil dari Inspect Element > Copy Selector.
  • • TEXT: Mode cerdas untuk mencari elemen berdasarkan teks yang terlihat. Sangat berguna jika ID elemen berubah-ubah (dynamic ID).
  • • COORD: Fitur presisi tingkat tinggi. Mengarahkan kursor dan melakukan klik pada koordinat (X,Y). Gunakan jika elemen berada di dalam HTML5 Canvas atau tertutup layer transparan.

🛠️ Self-Healing Engine (Retry)

Fitur ini memberikan kemampuan Auto-Recovery pada script. Jika dicentang, script tidak akan langsung mati (crash) saat elemen tidak ditemukan, melainkan akan menunggu dan mencoba ulang sesuai jumlah Max Try yang Anda tentukan.

🚀 Turbo Mode

Mengaktifkan Request Interception. Script akan memblokir download gambar, font, dan media. Hasilnya: Loading web 3x lebih cepat dan penggunaan RAM di RDP jauh lebih hemat.

03 Deployment Sequence

Ikuti prosedur ini untuk menjalankan bot Anda:

1. Susun langkah kerja pada Builder Panel (sebelah kiri).

2. Klik Compile Stealth Engine untuk memproses logika.

3. Copy seluruh output kode dan simpan sebagai file bot.js.

4. Jalankan payload menggunakan engine Node.js:

node bot.js
COPY
